## React-vote

> react web app

###  技术栈
react + react-router + axios + mobx + mobx-react + antd-mobile

## 目录结构
```
react-vote/
   |
   ├──icons/                    * 用到的icon图标
   |
   ├──components/               * 可复用组件
   │
   │──pages/                    * 展示的页面
   |
   ├──store/                    * mobx
   │
   │──axios/                    * axios的配置文件
   │
   │──utils/                    * 工具函数
   │    |
   |    |—— db.js               * localstorage操作
   |    |
   |    |__ index.js            * 别的工具函数
   │
   │──index.css                 * 全局的样式，其中也有修改了ant-mobile的样式
   │__index.js                  * 入口文件
```
